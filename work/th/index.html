Insight

대표 화면인 메뉴 목록은 크게
카트 하단형과 우측형 2가지로 나타났습니다.

레이아웃 2가지 유형

카드 하단 또는 우측으로 나뉜 2가지 레이아웃은
사용성에 차등이 있기 보다는 개인에 따라 다르게 느껴지는 정도입니다.
우측 레이아웃이 담은 상품에 대해 확인이 잘되는 측면이 있으나,
시선이 분산되어, 메뉴 집중도를 해치는 경향도 있습니다.

Solution

주문 서비스의 첫 화면은
다양한 필요를
빠르게 충족하도록.

작지만, 사용자와
함께하는 개인화 요소

주문 앱의 홈 화면에 꼭 필요한 요소를 정의하였습니다.
우선 주문 버튼을 화면 내에 눈에 띄게 두었습니다.
매장의 테이블에 있는 QR을 스캔하여 바로 주문하는 버튼과
키오스크 주문 시 사용하는 멤버십 바코드 조회 버튼도
바로 꺼내 써야하기에 홈 화면에 배치하였습니다.

• 효과적인 설명을 위해 디자인 이미지를 사용하였으나,
실제로는 와이어프레임과 기획 작업을 하였습니다.

개인화된 서비스는 홈 화면에도 볼 수 있습니다.
지난 주문을 간편하게 다시 주문하는 간편 주문으로
사용자 맞춤형 빠른 주문을 제공합니다.
닉네임을 활용한 그리팅 메시지와 사용자의 스탬프,
쿠폰 보유 현황을 상단에 표시하여, 사용자 관점으로
필요한 요소를 전달합니다.
