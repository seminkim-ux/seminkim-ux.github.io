UX 컨설팅 및 출입, 제어, 모빌리티, 커뮤니티로 구성된 상세화면 전반을 설계하였습니다.

Overview

개별 앱으로 분산된 IoT 기능을 통합.
IPARK만의 특색을 살린 UX 제시.

개편 전 아이파크의 IoT 서비스는 제어, 주차, 출입 등의 기능을 개별앱으로 제공하고 있었습니다.
이 기능을 하나의 앱으로 통합하며, 경쟁사인 푸르지오, 래미안과 차별점을 갖는 UX가 필요했습니다.
마켓 리서치와 임직원 인터뷰, 유저 리서치를 통해 UX콘셉트 및 구체적인 실행 전략을 설정하였습니다.

Insight

사용자가 어려움을 겪는 Pain Point를 해결할
4가지 UX 전략을 세우고 구체적으로 실행하였습니다.

UX Writing

  • 지시어나 단어의 나열이 아닌
    구어체로 친근하게 안내
  • 문장은 간결하게 정리하여
    의미파악이 명확하도록

규칙성

  • 규칙성을 가진 화면구성으로
    사용자가 충분히 익숙해지도록
  • 버튼명, 내비게이션 구조도
    규칙성을 갖도록

사용자 상태, 맥락

  • 사용자에게 공감받으려면
    먼저 사용자를 살펴야 함
  • 사용자 상태나 맥락상 필요한
    정보를 제공

사용보다
앞선 이해

  • 사용자의 이해를 돕는
    콘텐츠 제공 필요
  • 에플, 네이버 등에서도
    신규 서비스나 고도화 기능은
    콘텐츠를 제공함

Solution

UX Writing

친근한 UX Writing으로
사용자에게 다가갑니다.

실제로는 와이어프레임과 UX정책 작업을 하였습니다.
