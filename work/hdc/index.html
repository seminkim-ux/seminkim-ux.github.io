HDC IoT App

현대산업개발 IoT 제어서비스 앱 IPark 스마트홈의 UX를 1, 2차에 걸쳐 기획하였습니다.

UX 컨설팅 및 출입, 제어, 모빌리티, 커뮤니티로 구성된 상세화면 전반을 설계하였습니다.

화면별 인터랙션과 부가 기능도 기획하였습니다.

ROLE UX PLANNING / 90% PROGRESS Increased 10% MAU

Overview

개별 앱으로 분산된 IoT 기능을 통합.
IPARK만의 특색을 살린 UX 제시.

개편 전 아이파크의 IoT 서비스는 제어, 주차, 출입 등의 기능을 개별앱으로 제공하고 있었습니다.
이 기능을 하나의 앱으로 통합하며, 경쟁사인 푸르지오, 래미안과 차별점을 갖는 UX가 필요했습니다.
마켓 리서치와 임직원 인터뷰, 유저 리서치를 통해 UX콘셉트 및 구체적인 실행 전략을 설정하였습니다.

초기 예상보다 적용속도가 더딘 이유는 기기들이 서로 호환성되지 않는 기술적인 한계가 존재하며,
비용과 투자면에서 인프라와 구축 비용이 상당하기 때문입니다. 새로운 기술에 대해 사용자 수용성과 일상의 필요성도 낮았습니다.
다만 가속화되는 AI기술과 함께 더욱 보편화될 것으로 예상됩니다. IoT 데이터의 활용과 자동화는 생활의 변화를 더 빠르게 가져올 것입니다.

중간단계에 있는 현재는 익숙한 사용과 필요성을 어필하는 UX가 필요합니다.

다양한 기능을 통합하여 잘 제공한 후, 추후에는 아이파크몰과 연계한 세일즈까지 염두에 두었습니다.

Insight

사용자가 어려움을 겪는 Pain Point를 해결할
4가지 UX 전략을 세우고 구체적으로 실행하였습니다.

UX Writing

  • 지시어나 단어의 나열이 아닌
    구어체로 친근하게 안내
  • 문장은 간결하게 정리하여
    의미파악이 명확하도록

규칙성

  • 규칙성을 가진 화면구성으로
    사용자가 충분히 익숙해지도록
  • 버튼명, 내비게이션 구조도
    규칙성을 갖도록

사용자 상태, 맥락

  • 사용자에게 공감받으려면
    먼저 사용자를 살펴야 함
  • 사용자 상태나 맥락상 필요한
    정보를 제공

사용보다
앞선 이해

  • 사용자의 이해를 돕는
    콘텐츠 제공 필요
  • 에플, 네이버 등에서도
    신규 서비스나 고도화 기능은
    콘텐츠를 제공함

최근 사용 메뉴

사용자는 많은 메뉴 중 잘 사용하는 것만 계속 사용할
가능성이 높습니다. 사용한 기록이 없다면 표시하지 않지만,
사용 내역이 있다면 홈화면 상단에 먼저 제공합니다.

버튼 편집 기능

홈 화면 내 사용하지 않는 기능을 보이지 않도록
숨기거나 순서를 변경할 수 있습니다. 많은 기능
버튼을 편집하여 선택과 집중이 가능합니다.
