ROLE UX PLANNING / 100% PROGRESS -

Overview

디자인 및 사용성 업그레이드

NH 투자증권의 모바일 그룹웨어인 모바일 오피스는 2014년 처음 런칭되었습니다.
그 이후 화면과 기능들이 필요에 의해 계속해서 추가되었습니다. 시점이 다른 기능 중심의 추가된 화면들은 통일성이 떨어졌습니다.
2024년 통합 개편 프로젝트에서 디자인 통합과 사용성 업그레이드가 필요했습니다.
현시점의 유저 사용성을 고려한 UX 업그레이드를 제공했습니다.

Design Upgrade

  • NH투자증권 아이덴티티 적용
  • 2024년도에 맞는 디자인
  • 개별적으로 진행된 디자인을 통일

Usability Upgrade

접근 구조 개선

  • 하단 Navigation 버튼으로 주요메뉴 빠르게 이동
  • 추가/수정된 화면들의 버튼 중복 등 내비게이션 개선
  • 모든 기능보다 자주쓰는 메뉴 중심으로 버튼을 구성한 메인 화면
  • 모든 목록을 표시하기 전에 디폴트 접근을 우선(메일함, 결재함 등)

가독성 중심의 화면 개선

  • 집중을 방해하는 불필요한 버튼 제거
  • 노후화된 UI를 가독성 중심으로 개편

태블릿 사용성 업그레이드

  • 태블릿에 적합한 UI 적용
  • 넓은 화면에서 효과적인 시각적 구조화
  • 태블릿 전용 메뉴를 태블릿 내에서 더 강조

Insight

종합 포털에서 심플한 구성으로 변화

모바일 그룹웨어 UX는 포털 형태의 많은 기능과 정보를 한번에 담는 모습에서,
개인의 업무 효율을 위한 심플한 구성으로 변화하고 있습니다.

Portal

그룹웨어 초기, 최대한 많은 기능과 콘텐츠를
메인화면에 제공. 기업포털의 느낌을 강조.
모바일 초기 사이트 메인과 Microsoft의 Metro UI의 영향

Simple UI

사용경험이 쌓이며, 많은 정보와 기능의 나열보다 빠르게 접근하는 UI를 선호. (타사스템 사용자 조사 결과)
심플한 구글과 네이버 UI의 영향으로 간결한 화면을 선호하게 됨
주요 화면을 하단 Navigation으로 접근하고, 홈/소식 메뉴를 추가할 수 있는 개인화 적용.

개인 사용자가 원하는 메뉴로 수정할 수 있도록 함.

UX Concept

Approval

미결함 먼저 표시

모바일 결재는 미결함을 먼저 표시합니다.
다른 결재함 상태를 같이 확인할 수 있어
결재함 선택 화면이 별도로 필요없습니다.

다른 결재함 건수 표시

디폴트 미결함 외에 다른 결재함의 건수를
함께 표시하여, 한 눈에 확인하고 바로 이동합니다.
