Overview

Design and Usability Upgrade

NH Investment & Securities' mobile office, a mobile groupware, was first launched in 2014.
Since then, screens and features continued to be added as needed. Screens added at different times with a focus on functions lacked uniformity.
In the 2024 integrated redesign project, design integration and usability upgrades were necessary.
We provided UX upgrades considering current user usability.

Design Upgrade

  • Applied NH Investment & Securities identity
  • Design suitable for 2024
  • Unified designs that were implemented separately

Usability Upgrade

Access Structure Improvement

  • Quick navigation to main menus via bottom navigation buttons
  • Improved navigation including duplicate buttons on added/modified screens
  • Main screen configured around frequently used menus rather than all features
  • Prioritizing default access before displaying all lists (mail box, approval box, etc.)

Readability-Focused Screen Improvements

  • Removed unnecessary buttons that distract focus
  • Redesigned outdated UI with a focus on readability

Tablet Usability Upgrade

  • Applied UI appropriate for tablets
  • Effective visual structuring on larger screens
  • Greater emphasis on tablet-specific menus within tablet view

Insight

Shifting from Comprehensive Portal to Simple Configuration

Mobile groupware UX is changing from a portal-type layout that contains many features and information at once,
to a simpler configuration for individual work efficiency.

Portal

In early groupware, providing as many features and
content as possible on the main screen.
Emphasizing the corporate portal feel.
Influenced by early mobile site mains and Microsoft's Metro UI.

Simple UI

As user experience accumulated, users preferred UIs that provided quick access rather than displaying many features and information. (Based on third-party system user research)
The influence of simple Google and Naver UIs led to preferences for concise screens
Access to main screens via bottom navigation, with personalization allowing addition of home/news menus.

UX Concept

Mobile approval shows pending approvals first.

eliminating the need for a separate approval box
