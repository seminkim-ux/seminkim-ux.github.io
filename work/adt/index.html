ROLE UI DESIGN / 100% PROGRESS Service Launching

Overview

실시간 보안을 강조한 최초의 대고객 앱.

2015년 ADT캡스는 국내 탑 보안 업체였습니다.
실시간 보안 점검과 제어를 제공하는 최초의 온라인 통합 대고객 앱을 구축하여,
안전과 신뢰를 쌓는 새로운 고객 경험을 제공해야 했습니다.
대상 고객은 B2B 성격을 띄는 금융고객사와 B2C 일반 고객 2분류였습니다.

Solution

브랜드 아이덴티티를 활용한
서비스 효용성.

SK쉴더스로 사명 개편 전, ADT캡스는 파란 필각형을 브랜드 아이덴티티로 사용하고 있었습니다.
외부에 부착하는 현판과 출입기기의 점등 표시는 보안이 실시중 이라는 상징이었습니다.
이를 활용하여, 온라인에서도 확장된 통일성과 신뢰감을 부여하였습니다.

통일된 사인을 앱에서도.

출입기기 내 점등되는 사인을 앱에서 동일하게
보이도록 구성하였습니다. 이미 신뢰를 쌓은
아이덴티티를 온라인에도 적용하는 것이 주효했습니다.

Solution

전문성 있는 보안을
제공하는 B2B 메인.

금융 고객은 지점과 본점 2가지 유형으로 제공합니다.
각 은행 로고를 표시하며, 전문화된 서비스를 이용 중임을 느끼게 하였습니다.
본점에서는 각 지점별 상태를 명확하게 알 수 있도록 디자인하였습니다.

Solution

B2C 일반고객
내 손 안의 친근한 서비스

일반고객 메인화면 역시 자신의 소중한 재산을 안전하게 보호하는
경험을 브랜드 아이덴티티를 유지하며 디자인하였습니다.
전체 메뉴는 그룹핑을 명확하게 한 후, 사용도와 중요도에 맞게 제공하였습니다.
