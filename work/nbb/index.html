Solution

주문 서비스의 첫 화면은
다양한 필요를
빠르게 충족하도록.

작지만, 사용자와
함께하는 개인화 요소

주문 앱의 홈 화면에 꼭 필요한 요소를 정의하였습니다.
우선 주문 버튼을 화면 내에 눈에 띄게 두었습니다.
매장의 테이블에 있는 QR을 스캔하여 바로 주문하는 버튼과
키오스크 주문 시 사용하는 멤버십 바코드 조회 버튼도
바로 꺼내 써야하기에 홈 화면에 배치하였습니다.

개인화된 서비스는 홈 화면에도 볼 수 있습니다.
지난 주문을 간편하게 다시 주문하는 간편 주문으로
사용자 맞춤형 빠른 주문을 제공합니다.
닉네임을 활용한 그리팅 메시지와 사용자의 스탬프,
쿠폰 보유 현황을 상단에 표시하여, 사용자 관점으로
필요한 요소를 전달합니다.

Solution

빠르게 가입하고
로그인하여
진입 장벽을 낮게.

SNS 간편 가입 시, 정보 제공 동의 버튼 선택만으로
입력 없이 빠르게 가입할 수 있습니다.
자체 회원가입을 한다면, 로그인 아이디를 찾는 고민이 없도록
아이디는 휴대폰번호로 정하였습니다.
가입할 때 작성하는 정보도 최소화하였습니다.
휴대폰 번호 변경에 따른 방안도 마련하였습니다.

Solution

편안한 주문이
빠른 주문.

사용자가 주문 과정에서 필요한 요소를
먼저 제공하여 진입장벽을 낮춥니다.
고려해야 할 요소, 직접 선택해야 할 요소를
줄이면서, 안내도 충실히 제공하는
편안한 주문이 서비스와 사용자가 함께
동의를 이루어가는 필수적인 UX입니다.

1. 기존 배달지와 현재 위치를 비교하고
물어보아서, 오류가 없도록 합니다.

2. 로그인 하지 않을 때도, 주문 과정을 모두 경험 후,
최종 단계에서 회원/비회원 주문을 선택하도록 합니다.

3. 지난 주문이나 미리 설정한 간편주문으로
더 쉽게 주문할 수 있습니다.
